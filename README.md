# Webpack + ES2015 のサンプル

## このファイル群の作り方

1. npm モジュールの準備

````
$ npm init
$ npm i webpack babel-loader html-loader babel-core  babel-preset-es2015 --save
````

2. .gitignore ファイルを作成

3. src ディレクトリに JSファイルを作成

4. `webpack.config.js` を作成

5 `$ ./node_modules/.bin/webpack `でビルド
